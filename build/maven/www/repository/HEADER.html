Maven repository for Geotools binaries and dependencies

This directory contains Geotools binaries and some dependencies not found in the Ibiblio public repository. This is a complement (not a replacement) for Ibiblio. The directory layout is understood by Maven 2, a project management tool used by the Geotools project. Geotools users are free to link their own project to this repository in order to get automatic downloads of Geotools binaries. A tutorial for doing so is provided there.
