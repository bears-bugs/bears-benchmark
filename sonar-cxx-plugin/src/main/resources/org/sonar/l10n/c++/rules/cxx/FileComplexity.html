The Cyclomatic Complexity is measured by the number of (&&, ||) operators and (if, while, do-while, for, ?:, catch, case, default) statements in the source file plus one for each function.

Even when the Cyclomatic Complexity of a file (class) is very high, this complexity might be well distributed among all functions. Nevertheless, most of the time, a very complex file is a file which breaks the Single Responsibility Principle and which should be re-factored to be split in several files (classes).
