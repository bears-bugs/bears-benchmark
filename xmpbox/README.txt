====================================================
Apache XmpBox <http://pdfbox.apache.org/>
====================================================

The Apache XmpBox library is an open source Java tool that implements 
Adobe's XMP(TM) specification.

You need Java 6 (or higher) and Maven 2 <http://maven.apache.org/> to
build XmpBox. The recommended build command is:

    mvn clean install

The default build will compile the Java sources and package the binary
classes into a jar package.

See the Maven documentation for all the other available build options.

See the issue tracker at https://issues.apache.org/jira/browse/PDFBOX 
(component XmpBox) for the full list of known issues and requested features.

XmpBox is a subproject of Apache PDFBox. PDFBox is a project of the
Apache Software Foundation <http://www.apache.org/>.
