==================================
CorfuDB - A Distributed Shared Log 
==================================

.. image:: _static/corfu.png              
   :scale: 40 %
   :align: left

Corfu is a consistency platform designed around the abstraction of a shared log. CorfuDB objects are in-memory, highly available data structures providing linearizable read/write operations and strictly serializable transactions. CorfuDB is based on peer-reviewed research.
