=====

    Yet another Java library for Next Generation Sequencing (NGS) data processing
    
## Usage

**Maven:**
```
<dependency>
    <groupId>com.milaboratory</groupId>
    <artifactId>milib</artifactId>
    <version>1.8.4</version>
</dependency>
```
